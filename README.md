## 1. Advanced ATM Simulation
```
Description:
Simulates an ATM machine with multiple users, allowing operations like withdraw, deposit, check balance, and PIN change.

Features:

User login with account number and PIN

Check balance, deposit money, withdraw money

Transaction history

Admin can add/remove accounts

OOP Concepts Used:

Classes & Objects (User, Account, ATM)

Inheritance (Admin extends User)

Encapsulation (private fields, getters/setters)

Polymorphism (method overloading for transactions)

Working Flow:

User logs in with account number and PIN.

Display menu: Check Balance, Deposit, Withdraw, Exit.

Perform selected operation.

Update balance and store transaction history.

Loop until user exits.
```
